# Control Structures - If Statement
```javascript
var condition = true;

if (condition) {
    // true
    console.log('Executed!'); // Executed!
} else {
    // false
    console.log('Not Executed!');
}


// if else if
var condition2 = false;
var condition3 = true;

if (condition2) {
    // true
    console.log('Executed!');
} else if (condition3) {
    // true
    console.log('Still Executed!'); // Still Executed!
} else {
    // false
    console.log('Not executed!');
}

```
