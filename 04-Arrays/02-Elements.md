# Basics & Managing Elements
```javascript
var array = [1, 2, 3];
console.log(array.length); // 3
console.log(array[1]) // 2

array[1] = 100;
console.log(array); // [1, 100, 3]

array [3] = 100;
console.log(array); // [1, 100, 3, 100]

array[5] = 100;
console.log(array); // [1, 100, 3, 100, undefined, 100]

```
